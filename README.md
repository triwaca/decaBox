# printedOrganizer3D
My solution for small 3D printed drawers and organizers using less filament.

- A complete set with two drawers can be printed with 130 grams! Both drawers and body must use the Vase option on your 3D printer (with line width 0.8).

![3D printed drawers](images/3dOrganizer1.jpg)

- Drawer can hold default component packages and boards.

![Drawer with bags of components](images/matchComponents.jpg)

- You can remove the drawer and use the handle to pour components, like screws.

![Easy pouring components from drawer](images/easyPouring.jpg)

- Vase mode allows fast printing, and save lots of filament! Each drawer uses only 22 grams!

![Printing in Vase mode](images/printingVaseMode.gif)
